## Requirements
 - Local (MAMP, WAMP etc.) or remote HTTP server with PHP.

## Setting Up
 - Download and unzip on local or remote server.
 - Go to '/assets/inc/pages_config.php' and setup "base_url" variable.
 - Done! You're ready to write some code...
